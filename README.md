# Retinal Blood Vessel Segmentation and Disease Prediction using U-Net and CNN on NVIDIA Jetson Nano

## Overview

Retinal diseases such as Diabetic Retinopathy and Glaucoma are among the leading causes of vision impairment worldwide. Early diagnosis can significantly improve treatment outcomes and prevent permanent vision loss. This project presents an automated retinal image analysis system that combines Deep Learning and Edge AI technologies to perform retinal blood vessel segmentation and retinal disease prediction.

The proposed system utilizes a U-Net architecture for accurate retinal vessel segmentation and a Convolutional Neural Network (CNN) for disease classification. The complete solution is deployed on the NVIDIA Jetson Nano platform, enabling real-time inference for portable and low-cost healthcare applications.

## Key Features

- Retinal Blood Vessel Segmentation using U-Net
- Retinal Disease Prediction using CNN
- NVIDIA Jetson Nano Deployment
- DRIVE Dataset (Kaggle) for Vessel Segmentation
- FIVES Dataset (Kaggle) for Disease Classification
- R4V2025 Real-Time Hospital Dataset for Validation
- Real-Time Edge AI Healthcare Solution
